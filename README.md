# บทนำสู่ Data Mining

## ทำไมต้อง Data Mining?
ปัจจุบันข้อมูลมีปริมาณมากจากธุรกิจ วิทยาศาสตร์ เว็บ หรือระบบดิจิทัลต่าง ๆ  
ปัญหาคือ “ข้อมูลมีเยอะ แต่ความรู้ที่ใช้ประโยชน์ได้มีน้อย”  
จึงต้องใช้ Data Mining เพื่อช่วยวิเคราะห์และค้นหารูปแบบสำคัญจากข้อมูลจำนวนมาก

---

## Data Mining คืออะไร?
การค้นหารูปแบบหรือความรู้ที่มีประโยชน์จากข้อมูลขนาดใหญ่  
เป็นส่วนหนึ่งของกระบวนการ KDD (Knowledge Discovery in Databases)  
เน้นหาสิ่งที่ “ไม่ชัดเจน” และ “ไม่เคยรู้มาก่อน”

---

## กระบวนการ KDD
1. ทำความสะอาดข้อมูล (Cleaning)  
2. รวมข้อมูลจากหลายแหล่ง (Integration)  
3. เลือกข้อมูลที่ต้องใช้ (Selection)  
4. ทำ Data Mining เพื่อค้นหารูปแบบ  
5. ประเมินความน่าสนใจของรูปแบบ (Evaluation)  
6. นำเสนอผลลัพธ์ (Presentation)

---

## มุมมองของ Data Mining
- ประเภทข้อมูลที่ใช้ได้: ฐานข้อมูล ข้อความ เว็บ ภาพ วิดีโอ กราฟ เครือข่ายสังคม  
- รูปแบบที่ค้นหา: การจำแนก การจัดกลุ่ม ความสัมพันธ์ ค่าผิดปกติ แนวโน้ม  
- เทคโนโลยีที่เกี่ยวข้อง: Machine Learning, สถิติ, OLAP, Visualization  
- การใช้งาน: โทรคมนาคม การเงิน การแพทย์ การตลาด เว็บ

---

## ฟังก์ชันหลักของ Data Mining
- **Generalization** – สรุปข้อมูล  
- **Association** – ความสัมพันธ์ของรายการ  
- **Classification** – จำแนกข้อมูล  
- **Clustering** – จัดกลุ่ม  
- **Outlier Detection** – ค่าผิดปกติ  
- **Trend & Sequence** – ลำดับและแนวโน้ม  
- **Graph & Network Mining** – วิเคราะห์เครือข่าย

---

## ตารางสรุปฟังก์ชันของ Data Mining

| ฟังก์ชัน | ความหมาย | ตัวอย่าง |
|---------|-----------|----------|
| Generalization | สรุปรายละเอียดให้เห็นภาพรวม | รายงานสรุปยอดขาย |
| Association | หาว่ารายการใดเกิดร่วมกันบ่อย | A → B |
| Classification | จำแนกข้อมูลตามกลุ่มที่กำหนด | spam / not spam |
| Clustering | จัดกลุ่มข้อมูลอัตโนมัติ | แบ่งกลุ่มลูกค้า |
| Outlier Detection | หาค่าที่แตกต่างจากกลุ่ม | ตรวจการทุจริต |
| Trend / Sequence | วิเคราะห์ข้อมูลตามเวลา | แนวโน้มยอดขาย |
| Graph Mining | วิเคราะห์โครงสร้างความเชื่อมโยง | โครงสร้างเว็บ |

---

## ประเด็นท้าทาย
- ข้อมูลมีขนาดใหญ่และหลายรูปแบบ  
- มี noise และข้อมูลไม่สมบูรณ์  
- ต้องการประมวลผลที่รวดเร็ว  
- ต้องคำนึงถึงความเป็นส่วนตัวและความปลอดภัย

---

## ภาพรวมงานวิจัยและการใช้งาน
มีการพัฒนาอย่างต่อเนื่องผ่านงานประชุม เช่น KDD, ICDM, PAKDD  
องค์กรใหญ่ใช้ Data Mining ในระบบค้นหา แนะนำสินค้า วิเคราะห์ผู้ใช้ ตรวจสอบความผิดปกติ ฯลฯ
